# Event-Day Checklist (Dynamic Engine)

## T-60
1. Verify `/health`.
2. Admin login and configure event profile.
3. Validate generated paths, rooms, and trap count via monitor.
4. Create all team credentials.
5. Export room codes and generate/print QR cards.
6. Print and place mandatory final-key QRs:
   - `EVENT_ID-FINAL-KEY-NEXUS`
   - `EVENT_ID-FINAL-KEY-AMIPHORIA`
   - `EVENT_ID-RAPID-FIRE-QR`

## T-30
1. Dry run one team end-to-end.
2. Validate adaptive traps fire for fast team profile.
3. Validate hints and penalties.
4. Confirm leaderboard hidden.
5. Confirm websocket monitor live updates.

## T-10
1. Keep admin dashboard open.
2. Confirm Wi-Fi and camera permissions.
3. Ensure support fallback device with admin token ready.

## During event
1. Watch room occupancy heatmap for congestion.
2. Watch suspicious activity feed.
3. Use disqualify action only with manual adjudication.

## End
1. Reveal leaderboard.
2. Export logs.
3. Archive event config + leaderboard snapshot.
